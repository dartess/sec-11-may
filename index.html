Цвет в компьютере

Изображение состоит из пикселей

Пиксель состоит из трёх частей

Цветовая модель RGB

  • R — red
  • G — green
  • B — blue

Любой цвет составляется комбинацией трёх компонент.

Каждая компонента может принмиать значение от 0 до 255, итого 256 вариантов.

Благодаря большому количеству комбинаций, пиксель может принимать 256 * 256 * 256 = 16 777 216 цветов.

Цветовая модель RGB

0 — минимальное значение компоненты (отсуствие свечения цвета)

255 — максимальное значение компоненты (максимальное свечение цвета)

rgb()

Функция, которая возвращает цвет, составленный из трёх компонент.

0 — минимальное значение компоненты (отсуствие свечения цвета)

255 — максимальное значение компоненты (максимальное свечение цвета)

rgb(0, 0, 0) —      

rgb(255, 0, 0) —      

rgb(0, 0, 255) —      

rgb(127, 0, 0) —      

rgb(255, 0, 255) —      

rgb(255, 255, 255) —

#HEX

Каждая компонента записывается числом от 00 до FF (от 0 до 255).

Код пишется после решётки.

Код может содержать либо 6, либо 3 цифры. Если цифр 3, каждая дублируется.

Регистр не имеет значения.

#HEX: примеры

color: rgb(255, 0, 0)

color: red

color: #FF0000

color: #ff0000

color: #f00

Прозрачный цвет (альфа-канал)

Цвет можно указать со степенью прозрачности.

Для этого используется другая функция: rgba(red, green, blue, opacity).

Opacity — степень прозрачности, от 0 до 1.

color: rgba(255, 0, 0, 0.5)

color: rgba(255, 0, 0, .5)

color: #FF000080
